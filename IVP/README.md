IVP I ("sim28"), IVP II ("sim32"), and IVP III ("sim27") may be numerically solved by running

    python IVP_I.py
    python IVP_II.py
    python IVP_III.py

Using the full data output from these simulations, the animations "fig2.mp4" and "fig4.mp4" (available on our [Zenodo repository](https://doi.org/10.5281/zenodo.18357092) at [doi:10.5281/zenodo.18357092](https://doi.org/10.5281/zenodo.18357092)) may be generated using `plot_sincos.py`

    python plot_sincos.py sim28
    python plot_sincos.py sim32

The additional animation "supplementary.mp4" on the [Zenodo repository](https://doi.org/10.5281/zenodo.18357092) requires both the full data from IVP III ("sim27") and data generated by the scripts in `EVP/`. If both data sources are available, the animation frames may be generated using

    python plot_cg.py sim27

Simulation snapshots and one-dimensional timeseries data may be generated from the full simulation data using `export.py` and `export_timeseries.py`, respectively. E.g.,

    python export.py sim28
    python export_timeseries.py sim28

Alternatively, these snapshots and timeseries may be downloaded from the [Zenodo repository](https://doi.org/10.5281/zenodo.18357092) and placed at `IVP/data`.<br>
From `IVP/`, run<br>

    wget -L -O data.zip "https://zenodo.org/records/18357093/files/data.zip"
    unzip data.zip
    rm data.zip
    
Using these snapshots, the simulation results figures in the manuscript may be generated with `plot_ivp.ipynb`.
